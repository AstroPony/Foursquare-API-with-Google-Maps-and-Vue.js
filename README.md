# Adyen Challenge

## Demo

[challenge.ratemylerao.co.za](https://challenge.ratemylerao.co.za)

## Build Setup
``` bash
# install dependencies
npm install

# run on local (localhost:8080)
npm run dev

# production build with minification
npm run build

# production build with server
npm run start
```
